# Record Work Skill

Use this after finishing work on an OpenSpec change.

## Steps

1. Create or update a wiki page in `agent-wiki/` using the `llm-wiki` skill.
2. Use this naming: `YYYY-MM-DD-<change-folder-name>.md`
3. The wiki page must include these sections:

   - **Date**
   - **OpenSpec Change** (link to the folder)
   - **Tasks Completed** (from `tasks.md`)
   - **Verification Against Spec** (Detailed check: Did it pass all requirements? What was the result of each major acceptance criteria?)
   - **Key Decisions**
   - **Problems Encountered & Solutions**
   - **Current Status** (In Progress / Ready for Review / Ready to Archive)
   - **Recommended Next Steps**

4. Update `agent-wiki/index.md` if needed.
5. Be honest and precise in the verification section.
